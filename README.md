# social_network_system_design
System Design социальной сети для курса по System Design 13

Функциональыне требования:

      -Публикация постов с фотографиями, описанием и привязкой к месту путешествия
      
      -Оценка и комментарии постов других путешественников
      
      -Подписка на других путешественников
      
      -Поиск популярных мест для путешествий и просмотр постов с этих мест
      
      -Просмотр ленты других путешественников и ленты пользователя в обратном хронологическом порядке

Нефункциональные требования:

      -DAU 10 000 000
      
      -Геораспределение: страны СНГ
      
      -Сезонность: ×2 нагрузка (май-август)
      
      -Доступность: 99.9%


Нагрузка:

    Активности на пользователя в день:
    
      -Создание постов: 0.1 (10% пользователей создают 1 пост)
      
      -Просмотр ленты: 50 раз
      
      -Комментарии: 0.5 (1 комментарий каждым вторым пользователем)
      
      -Лайки: 5
      
      -Подписки: 0.2
      
      -Поиск: 2 раза


RPS (с учетом сезонности ×2):

    Write RPS:
    
        -Посты: (10M × 0.1 × 2) / 86400 ≈ 23 RPS
        
        -Фото: (10M × 0.1 × 3 × 2) / 86400 ≈ 69 RPS
        
        -Комментарии: (10M × 0.5 × 2) / 86400 ≈ 116 RPS
        
        -Лайки: (10M × 5 × 2) / 86400 ≈ 1157 RPS
        
        -Подписки: (10M × 0.2 × 2) / 86400 ≈ 46 RPS
    
    Read RPS:
    
        -Лента: (10M × 50 × 2) / 86400 ≈ 11574 RPS
        
        -Поиск: (10M × 2 × 2) / 86400 ≈ 463 RPS

Traffic:

  - Пост:
    - user_id 8b
    - created_at 8b
    - discription 200b
    - geo tag 8b
    Total: 224b 

  - Фото:
    - image_id 8b
    - post_id 8b
    - image 2Mb
    Total: 2mb

  - Комментарий:
    - user_id 8b
    - comment_id 8b
    - post_id 8b
    - comment 200b
    Total: 224b 

Traffic_write:

      -Посты: 23 × 224b = 5.0 KB/s
      
      -Фото: 69 × 2MB = 138 MB/s
      
      -Комментарии: 116 × 224b = 25.4 KB/s
      
      -Лайки: 1157 × 16b = 18.1 KB/s
      
      -Подписки: 46 × 16b = 0.7 KB/s

Traffic_read:

      -Лента: 11574 × 20 постов × (224b + 2MB) ≈ 46.3 GB/s
      
      -Поиск: 463 × 50KB = 23.2 MB/s
